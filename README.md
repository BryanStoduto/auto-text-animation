# 💼 Auto Text Effect

Uma aplicação desenvolvida com **HTML5, CSS3 e JavaScript** que exibe um efeito de digitação automática, alternando entre diferentes profissões.

## 🚀 Funcionalidades

* Efeito de digitação letra por letra.
* Alternância automática entre diferentes profissões.
* Loop infinito após exibir todas as opções.
* Atualização dinâmica do conteúdo da página.

## 🛠️ Tecnologias

* HTML5
* CSS3
* JavaScript (ES6)

## 📌 Como funciona

O JavaScript percorre um array contendo diferentes profissões e exibe cada uma delas letra por letra. Quando uma profissão é completamente escrita, a próxima é exibida automaticamente. Ao chegar ao final da lista, o ciclo recomeça do início.

## ▶️ Como executar

1. Clone este repositório.
2. Abra o arquivo `index.html` no navegador.
3. Observe o efeito de digitação acontecendo automaticamente.
