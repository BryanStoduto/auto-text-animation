# 💼 Auto Text Effect

A simple application built with **HTML5, CSS3, and JavaScript** that creates an automatic typing animation, cycling through different career titles.

## 🚀 Features

* Letter-by-letter typing animation.
* Automatically switches between different career titles.
* Infinite loop through all career titles.
* Dynamic text updates without reloading the page.

## 🛠️ Technologies

* HTML5
* CSS3
* JavaScript (ES6)

## 📌 How It Works

The application stores career titles in an array and displays each one using a typing effect. Once a title is fully typed, the next one is displayed automatically. After the last title, the animation restarts from the beginning, creating an infinite loop.

## ▶️ Getting Started

1. Clone this repository.
2. Open the `index.html` file in your web browser.
3. Watch the automatic typing animation in action.
